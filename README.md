# 犬そだて

子犬を拾って伝説の犬まで育てる、シミュレーション系育成Webアプリ。

- 5段階進化（こいぬ → わんこ → 立派な犬 → 忠犬 → 伝説の犬）
- 緩めのレベルカーブ（Lv 1/3/8/18/35 で進化）
- 各アクションでキャラがアニメ（もぐもぐ / ぴょん / フルフル / ぎゅっ）
- プロップドロップ演出（🍖 🎾 🦴 💕）
- LocalStorage セーブ、オフライン経過反映（24h上限）
